# PHQ-9-Analysis
Exploratory Data Analysis of Student Depression Severity using PHQ-9 Scores and Related Psychosocial Factors with Python

Proyek ini merupakan analisis eksploratif untuk memahami tingkat keparahan depresi pada siswa dan mahasiswa menggunakan skor PHQ-9. Analisis ini bertujuan untuk mengidentifikasi pola gejala depresi serta keterkaitannya dengan faktor psikososial berdasarkan data survei kesehatan mental mahasiswa.

## Tujuan
- Menganalisis distribusi tingkat keparahan depresi berdasarkan skor PHQ-9  
- Mengidentifikasi pola depresi berdasarkan karakteristik demografis  
- Mengeksplorasi hubungan antara depresi dan faktor psikososial

- Python  
- Pandas

## Catatan
Analisis ini bersifat eksploratif dan bertujuan untuk pembelajaran serta pengembangan portofolio. Hasil analisis tidak dimaksudkan sebagai alat diagnosis klinis.
